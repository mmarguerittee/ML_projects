# ML_projects
учебные проекты:
- Страховой риск (CatBoost)
задача бинарной классификации (прогноз страхового случая)
предобработка данных, работа с дисбалансом классов
подбор порога по F1-метрике для улучшения качества предсказаний
- Прогноз курения (LightGBM)
бинарная классификация с использованием LightGBM
feature engineering (BMI, липидные и медицинские индексы и их отношения)
кросс-валидация (StratifiedKFold), ранняя остановка по AUC
- Прогноз стоимости недвижимости (KNN Regression)
полный pipeline обработки сырых данных (парсинг текста, пропуски, категории)
feature engineering из текстовых полей (метро, площадь, дом, инфраструктура)
обучение KNNRegressor с подбором гиперпараметров через GridSearchCV
